AidBot
********************************************************************************************************************************
This bot is an open source, work in progress discord bot to allow fellow programmers to learn about GitHub contributions,
asyncronous programming and working with different API's using C#.

DO NOT modify the Program.cs class unless you know completely how it functions.

Feel free to modify other classes within the program, as long as classes and tasks are named appropriately. We don't mind if you wish to 
modify another developers code, but please refrain from deleting features unless its absolutely necessary.

Every feature you've added or every block of code you've modified please comment your name on the top and leave comments explaining the
feature(s).
